# Bachelor Projects - AI & API

This repository contains three AI-powered projects that leverage APIs and artificial intelligence technologies. One of these projects will serve as my Bachelor's thesis project.

## Projects

### 1. AI-Powered Code Assistant / Code Review Tool (Bachelor Project)
An intelligent code assistant that uses AI to help developers write better code and perform automated code reviews.

**Key Features:**
- Automated code review with AI-generated suggestions
- Code quality analysis and recommendations
- Integration with popular IDEs and version control systems
- Natural language code explanations

**Location:** [`/ai-code-assistant`](./ai-code-assistant)

---

### 2. Customer Support Chatbot with RAG
A customer support chatbot that uses Retrieval-Augmented Generation (RAG) to provide accurate and context-aware responses.

**Key Features:**
- RAG-based response generation
- Knowledge base integration
- Multi-turn conversation support
- Context-aware query understanding

**Location:** [`/customer-support-chatbot`](./customer-support-chatbot)

---

### 3. AI-Driven Business Intelligence Dashboard
An intelligent business analytics dashboard that uses AI to provide insights and predictions from business data.

**Key Features:**
- AI-powered data analysis and visualization
- Predictive analytics and forecasting
- Natural language querying
- Automated report generation

**Location:** [`/bi-dashboard`](./bi-dashboard)
