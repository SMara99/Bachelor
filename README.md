# Animal Recognition Project

## Overview

A bachelor project to recognize animals using CLIP embeddings and Qdrant vector database

## Perequisites

- Python
- Qdrant
- Dataset of animal images

## Installation

1. Clone the repository
2. Navigate to the project : cd animal-recognition
3. Create virtual environment : python -m venv venv
4. Activate virtual environment : .\venv\Scripts\Activate
5. Install dependencies : pip install -r requirements.txt
